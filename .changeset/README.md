# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in our repository](https://github.com/changesets/changesets)

We have a quick list of common questions to get you started engaging with this project in
[our FAQ](https://github.com/changesets/changesets/blob/main/docs/FAQ.md)

The basic workflow is:

1. Make your changes
2. Run `bun changeset` (or `bunx changeset`) to create a changeset - this will ask you which packages are affected and what type of change it is (patch/minor/major)
3. Commit the generated `.md` file along with your changes
4. When the PR is merged, a "Version Packages" PR will be created automatically
5. When that PR is merged, packages will be published to npm automatically

The overall flow is `bun changeset` -> commit -> PR -> merge -> "Version Packages" PR -> merge -> publish
